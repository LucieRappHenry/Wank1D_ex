Format des fichiers .mat
Tous les fichiers sont de "simples matrices" de données. Pas d'entête, pas de caractère, que des nombres !

L'année 2005 est incomplète, avec un début de periode différent suivant le type données (c'est pourquoi les fichiers n'ont pas le même nombre de lignes)

Les colonnes sont les suivantes :

FluxEC
C1 : Année
C2 : Mois
C3 : Jour
C4 : Heure
C5 : Minute
C6 : DoY decimal
C7 : H (W.m-2)
C8 : LE (W.m-2)

FLUxNRJ
C1 : Année
C2 : Jour julien
C3 : Heure
C4 : Minute
C5 : DoY decimal
C6 : Rn (W.m-2)
C7 : Albedo (-)
C8 : flux G n°1 (W.m-2)
C9 : flux G n°2 (W.m-2)
C10: flux G n°3 (W.m-2)
C11: SWin (W.m-2)
C12: SWout (W.m-2)
C13: LWin (W.m-2)
C14: LWout (W.m-2)

Tempsol et SMC
C1 : Année
C2 : Jour julien
C3 : Heure
C4 : Minute
C5 : DoY decimal
C6 : obs à 10cm (°C ou m3.m-3)
C7 : obs à 50cm (°C ou m3.m-3)
C8 : obs à 100cm (°C ou m3.m-3)
C9 : obs à 150cm (°C ou m3.m-3)
C10: obs à 200cm (°C ou m3.m-3)
C11: obs à 250cm (°C ou m3.m-3)
